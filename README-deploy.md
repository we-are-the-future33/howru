# howru — Vercel 배포 파일

## GitHub repo에 올릴 파일 목록

루트 디렉토리에 다음 파일들 업로드:

### 핵심
- `index.html` (이전 howru.html)
- `vercel.json`
- `manifest.json` (PWA)

### 아이콘 (모두 루트에)
- `favicon-16.png`
- `favicon-32.png`
- `favicon-48.png`
- `apple-touch-icon.png` (180×180, iOS 홈화면)
- `icon-192.png` (PWA)
- `icon-512.png` (PWA)
- `og-image.png` (1200×630, SNS 공유 시)

### 선택사항
- `icon-1024.png` (고해상도 백업, 안 올려도 됨)
- `icon-master.svg` (소스 SVG, 안 올려도 됨)

---

## 배포 후 확인

1. https://howru-one.vercel.app 접속
2. 브라우저 탭에 favicon (구름 + 웃는 얼굴) 보이는지
3. 카카오톡/슬랙에 링크 붙여서 OG 미리보기 확인
4. 모바일 크롬에서 "홈 화면에 추가" → 아이콘 멋지게 나오는지
